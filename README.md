## Description

This project is a decentralized checklist application built using **Solidity, Ethers.js, and React (Next.js)**.  
It allows users to manage a personal on-chain checklist connected to their wallet, ensuring all data is transparently stored and verifiable on the blockchain.

Each checklist item is stored inside a smart contract deployed on a blockchain network. Users interact with their tasks directly through MetaMask, ensuring full ownership and decentralized control over their data.
